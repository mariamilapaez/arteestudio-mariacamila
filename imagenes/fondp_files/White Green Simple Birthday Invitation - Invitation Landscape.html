El editor de Canva tiene varias secciones: la navegación principal, el panel lateral, la barra de herramientas, el lienzo y los controles de página. Para enfocarte en una sección y conocer más sobre cada una, usa el tabulador. Para ir al lienzo, presiona "Ctrl+F2".
La navegación principal contiene botones con opciones para trabajar sobre el documento. Para pasar de un botón a otro, usa las teclas de flechas a la izquierda y a la derecha y, para activar un botón, usa la tecla Enter. Para ir al lienzo, presiona "Ctrl+F2".

El título del diseño tiene el nombre de tu diseño. Para editar el título del diseño, presiona la tecla Enter. Para dejar de editar el título del diseño, presiona la tecla Escape.
La barra de herramientas contiene botones para editar el elemento seleccionado. Para pasar de un botón a otro, usa las teclas de flechas a la izquierda y a la derecha y, para activar un botón, usa la tecla Enter. Para ir al lienzo, presiona "Ctrl+F2".
El lienzo es donde está tu diseño. Para ir a la página, presiona la tecla Enter. Para ir a grupos o elementos en la página, vuelve a presionar la tecla Enter. Presiona el tabulador para ir a la siguiente página, al siguiente grupo o al siguiente elemento. Para usar la barra de herramientas, usa la tecla "Ctrl+F1". Si hay un elemento seleccionado, puedes moverlo usando las teclas de flechas; puedes rotarlo con la teclas "," y "."; y puedes cambiar su tamaño con las teclas de flechas y la tecla "Ctrl". Puedes aumentar los incrementos de todos estos atajos con la tecla "Mayús".
